Dice Roll Game

It's a multiplayer game. Two players can play in this game. First player rolls dice. Each time player gets a number gets added to his current score. But if he gets 1 his current score becomes zero and game shifts to second player. So if player wants to secure the score he has to hold the game before getting one.
